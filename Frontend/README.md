# CodeGuru AI

A frontend-only Streamlit application designed as an interactive coding education platform for hackathon demos.

## Features

- Modern SaaS-style dashboard with learning analytics
- Learn Concepts browse experience with mock concept details
- AI Coding Coach chat simulator
- Bug Finder code analysis interface with mock issue detection
- Quiz Lab generator with multiple-choice questions
- Coding Challenges explorer with platform-style problem cards
- Progress Tracker with charts and skill analytics
- Settings for language, theme, and provider preferences

## Run the app

1. Install dependencies:

```bash
pip install -r Frontend/requirements.txt
```

2. Start the app:

```bash
streamlit run Frontend/app.py
```

## Localization

Supported languages:
- English (`en`)
- Telugu (`te`)

The locale files are available in `Frontend/locales/`.
